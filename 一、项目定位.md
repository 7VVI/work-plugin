# 一、项目定位

## 项目名称

Nav Portal（内部导航）

定位：

> 一个帮助开发团队统一管理内部系统、服务器、中间件、账号密码的 Chrome 浏览器插件。

主要特点：

- 一键打开内部系统
- 一个系统多个账号
- 自动填充账号密码（后续）
- 收藏服务器
- 收藏数据库
- Markdown 可读可编辑
- 数据本地存储
- 一键导入导出
- 后续支持团队同步

------

# 六、后续可扩展功能（V2）

这是整个产品最有价值的方向，建议在第一版预留扩展能力。

| 模块             | 功能                                                         |
| ---------------- | ------------------------------------------------------------ |
| 🔐 自动登录       | Content Script 自动识别登录页，一键填充账号密码，支持多账号切换 |
| 🌐 自动识别系统   | 打开任意网页时，自动匹配已保存系统，在工具栏显示"已收藏"状态 |
| 🖥 SSH 启动       | 与 Xshell、Tabby、Termius 等终端软件联动，一键发起 SSH 连接  |
| 🗄 数据库启动     | 一键打开 Navicat、DataGrip、DBeaver 等客户端并连接对应数据库 |
| 🧩 浏览器右键菜单 | "保存当前网站为内部系统"、"复制当前系统信息"等快捷操作       |
| ☁ 云同步         | 对接 WebDAV、Git、企业内部服务，实现多设备同步与版本管理     |
| 👥 团队共享       | 支持公共导航、团队标签、权限管理、只读/可编辑共享            |
| 🤖 AI 助手        | 基于导出的 Markdown 构建知识库，实现"Redis 测试环境在哪台服务器？""生产 RabbitMQ 登录账号是什么？"等自然语言查询 |
